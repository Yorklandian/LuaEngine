# LuaEngine
## 1.项目说明
    使用EasyX 2D图形库，将API的一些基本函数封装到lua的dll中以供lua调用。
    使用lua脚本调用上述dll以实现一个简单的lua脚本引擎API。用户可以利用这些API做一个简单的2D图像演示
## 2.安装
### 2.1只使用lua脚本
    如果对C代码部分不感兴趣，可以直接使用lua脚本和dll库。只需要克隆或者下载本项目即可。脚本的调用见使用教程。
